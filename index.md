## Hello
This is a tiny, **ultra‑lean** Jekyll starter. It uses a system font stack, a tiny ITCSS Sass setup, and zero JS by default.

- Edit this page at `index.md`
- Update site settings in `_config.yml`
- Styles live under `_sass/` and `assets/css/site.scss`

### Performance tips
- Keep images small and declare `width`/`height`
- Avoid third‑party embeds; use click‑to‑load if necessary
- Prefer text content; your users (and Core Web Vitals) will thank you
